# Rock Paper Scissors Game

A fun and interactive Rock Paper Scissors game built with vanilla JavaScript where you can play against the computer.

## Features

- Play against computer AI
- Score tracking for both player and computer
- Visual feedback for game results
- Interactive choice selection
- Real-time score updates
- Game status messages

## How to Play

1. Open `index.html` in your web browser
2. Click on your choice: Rock, Paper, or Scissors
3. The computer will make its choice automatically
4. See who wins each round based on the classic rules:
   - Rock beats Scissors
   - Scissors beats Paper
   - Paper beats Rock
5. Track your score against the computer's score

## Files

- `index.html` - Main HTML structure
- `app.js` - Game logic and computer AI
- `style.css` - Styling and layout
- `rock.png` - Rock choice icon
- `paper.png` - Paper choice icon
- `scissors.png` - Scissors choice icon

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## Running the Game

Simply open the `index.html` file in any modern web browser. No installation or build process required!
