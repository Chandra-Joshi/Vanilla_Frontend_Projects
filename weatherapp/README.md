# Weather App

A simple and elegant weather application built with vanilla JavaScript that displays current weather information for any city.

## Features

- Search weather by city name
- Display current temperature
- Show humidity percentage
- Display wind speed
- Clean and intuitive user interface
- Weather icon display

## How to Use

1. Open `index.html` in your web browser
2. Enter a city name in the search box
3. Click the search button to get weather information
4. View temperature, humidity, and wind speed for the selected city

## Files

- `index.html` - Main HTML structure
- `app.js` - Weather API integration and application logic
- `style.css` - Styling and layout
- `search1.jpg` - Search button icon
- `rains.jpeg` - Weather icon
- `humidity.png` - Humidity icon
- `wind.png` - Wind speed icon

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Weather API integration

## Running the Application

Simply open the `index.html` file in any modern web browser. Make sure you have an internet connection for the weather API to work properly.

## Note

You may need to configure your own API key in `app.js` to fetch live weather data.
